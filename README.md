# **CISM (Computer Inventory System Management)**

**Introduction**

CISM is a robust web-based system aimed at managing computer lab inventory. Designed for efficiency, it allows seamless tracking of hardware and software, ensuring smooth management of resources in computer labs.

![Screenshot 2024-12-01 233730](https://github.com/user-attachments/assets/4336615b-8c12-4776-822f-e01ab2c1c54a)

**Features/Uses**

- Add, edit, and delete inventory items.
- Monitor item availability and status.
- Role-based access (admin and regular users).
- Real-time inventory tracking.
- Report generation for inventory usage.

**Requirements**

- PHP 7.x or higher.
- MySQL database.
- Apache server (e.g., XAMPP).

**Installation**

1. Clone the repository via `git clone https://github.com/kents00/CISM.git`.
2. Install and configure XAMPP or a similar environment.
3. Import `computer_lab_inventory.sql` into your MySQL database.
4. Configure database credentials in `config/database.php`.
5. Access the project through `http://localhost/CISM`.
